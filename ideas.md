**Idea 1: Worn Naturalist**
- Design Movement: Analog zine meets botanical field guide
- Core Principles: Texture-first, imperfect-by-design, warm and earthy, hand-drawn energy
- Color Philosophy: Warm cream parchment (#F5EDD8), deep forest ink (#2C3A2E), dusty terracotta (#C4714A), muted sage (#7A9E7E). Colors evoke a worn journal, not a wellness app.
- Layout Paradigm: Offset asymmetric columns. Hero text bleeds left, imagery bleeds right. Sections use diagonal paper-tear dividers. No centered hero blocks.
- Signature Elements: Grain/noise texture overlay on all sections; hand-drawn dragonfly SVG motif; typewriter-style monospace for labels and callouts
- Interaction Philosophy: Hover reveals ink-bleed underlines; quiz cards flip like index cards; scroll triggers gentle paper-fold transitions
- Animation: 200ms ease-out for all transitions; quiz answer cards slide in from bottom-left with 40ms stagger; no bounce, no spring — deliberate and grounded
- Typography: Playfair Display (headings, bold, editorial) + Source Serif 4 (body, readable) + JetBrains Mono (labels, callouts)

**Idea 2: Quiet Brutalism**
- Design Movement: Swiss brutalism softened with organic warmth
- Core Principles: Bold type as structure, raw grid, no decorative flourishes, warmth through color not ornament
- Color Philosophy: Off-white (#FAFAF7), near-black charcoal (#1A1A18), deep amber (#D97706), pale clay (#E8DDD0). The amber is the only accent — used sparingly for CTAs and key phrases.
- Layout Paradigm: Full-bleed type sections. Large numbered steps. Oversized section labels in uppercase tracking. No cards — content lives directly on the page surface.
- Signature Elements: Thick horizontal rules as section dividers; oversized pull quotes that span the full column; amber highlight on key phrases mid-paragraph
- Interaction Philosophy: No decorative hover states — only functional ones. Quiz answers use large radio-style blocks, not cards. Buttons are rectangular, not rounded.
- Animation: Minimal. Sections fade in at 150ms. Quiz progress bar fills linearly. No entrance animations on scroll.
- Typography: Syne (headings, ultra-bold, geometric) + Lora (body, warm serif) + Syne Mono (labels)

**Idea 3: Soft Ink**
- Design Movement: Japanese wabi-sabi meets editorial wellness
- Core Principles: Negative space as content, ink-wash aesthetic, deliberate imperfection, calm authority
- Color Philosophy: Warm white (#FEFCF8), deep slate-teal (#2D4A4A), dusty rose (#C9897A), pale moss (#B5C4B1). The palette is quiet but not sterile — it feels like a well-used journal.
- Layout Paradigm: Single wide column with generous margins. Content floats left with a persistent right-margin annotation column for callouts and restart scripts. Feels like a marked-up manuscript.
- Signature Elements: Ink-wash watercolor dragonfly in the hero; annotation-style margin notes; thin ink-stroke dividers between sections
- Interaction Philosophy: Quiz flows as a single scrollable conversation, not a multi-step modal. Each question appears as the previous is answered, like a journal prompt.
- Animation: Questions slide up from below at 220ms ease-out; answered questions fade to 40% opacity; progress shown as a thin ink line growing left to right
- Typography: Cormorant Garamond (headings, elegant, high contrast) + Nunito Sans (body, readable, friendly) + Courier Prime (callouts, restart scripts)
</text>
